# TubeWatch™
## 유튜버를 위한 채널 성장 플랫폼

> **"열심히 올리는데 왜 안 늘까?"**
> 감이 아닌 데이터로 답을 찾아드립니다.

---

## TubeWatch란?

TubeWatch는 유튜브 채널 데이터를 분석해  
**다음에 무엇을 해야 할지** 명확하게 알려주는 성장 플랫폼입니다.

조회수, 구독자, 참여율 — 숫자만 보여주는 서비스가 아닙니다.  
**"지금 내 채널에서 무엇이 통하고, 다음에 무엇을 해야 하는가"** 까지 알려줍니다.

---

## 이런 분께 꼭 필요합니다

| 상황 | TubeWatch가 해결하는 것 |
|------|------------------------|
| 꾸준히 올리는데 구독자가 안 늘어요 | 성과를 막는 패턴을 찾아드립니다 |
| 어떤 주제를 올려야 할지 모르겠어요 | 내 채널 데이터 기반 다음 주제를 추천합니다 |
| 잘된 영상이 왜 잘됐는지 모르겠어요 | 반복되는 성공 공식을 추출합니다 |
| 유튜브 스튜디오 수치가 너무 복잡해요 | 핵심만 골라 한눈에 정리해드립니다 |
| 다음에 뭘 해야 할지 막막해요 | 오늘 바로 실행할 액션 리스트를 드립니다 |

## 핵심 기능 4가지

### 1. Channel Analysis — 채널 건강검진 리포트

> 내 채널이 지금 어떤 상태인지 한눈에

- 조회수·구독·참여율을 **하나의 점수**로 요약
- 잘된 영상 vs 안 된 영상 **차이 비교**
- 업로드 패턴 분석 — 최적 요일·시간대 파악
- 썸네일·제목이 클릭률에 미치는 영향 확인

```
💡 "내 채널 점수가 62점입니다. 업로드 일관성이 가장 큰 문제입니다."
```

---

### 2. Channel DNA — 성공 공식 추출기

> 내 채널에서 반복되는 '잘되는 이유'를 찾아드립니다

- 조회수 높은 영상들의 **공통점 자동 정리**
- 내 채널만의 강점과 핵심 방향성 파악
- 타겟 시청자층 분석
- 성과 구조 시각화 — 어떤 요소가 성과를 만드는지

```
💡 "귀하의 채널은 '문제 해결형 제목 + 10분 내외 포맷'에서 평균 2.3배 높은 성과가 나옵니다."
```

---

### 3. Action Plan — 오늘 당장 할 일 리스트

> 분석 결과를 실행 가능한 행동으로 바꿔드립니다

- 우선순위 순서로 정리된 **P1 ~ P3 실행 계획**
- 유튜브 스튜디오에서 **지금 바로 수정 가능한** 결손 항목
- 업로드 전 매번 확인할 체크리스트
- SEO 키워드 진단 및 태그 개선 제안

```
💡 "P1. 최근 5개 영상 설명란에 핵심 키워드가 빠져 있습니다. 지금 수정하세요."
```

---

### 4. Next Trend — 다음 영상 방향 제안

> 외부 트렌드가 아닌, 내 채널 내부 흐름에서 찾은 다음 주제

- 내 데이터 기반 **다음 영상 주제 후보 추천**
- 지금 가장 유력한 1순위 주제 강조 제시
- 최적 포맷 방향 (길이·형식) 권장
- 제목·훅·썸네일 방향까지 포함한 실행 힌트

```
💡 "지금 1순위: '초보자 실수 교정' 주제. 최근 3개 영상 중 이 패턴이 평균 1.8배 높은 반응을 보였습니다."
```

## 이렇게 사용합니다

```
① 채널 등록 (유튜브 채널 URL 입력, 1분 소요)
        ↓
② 채널 분석 실행 (AI가 최근 영상 자동 분석)
        ↓
③ 4개 리포트 확인
   ├ Channel Analysis — 현재 상태 진단
   ├ Channel DNA      — 성공 패턴 파악
   ├ Action Plan      — 오늘 할 일 실행
   └ Next Trend       — 다음 영상 방향 결정
        ↓
④ 영상 업로드 후 재분석 → 성장 확인
```

## TubeWatch가 다른 이유

**감이 아닌 근거** — 모든 제안은 내 채널 실제 데이터를 기반으로 합니다.  
**실행 중심** — 분석에서 끝나지 않고, 오늘 바로 할 일까지 알려줍니다.  
**내 채널 전용** — 다른 채널 평균이 아닌, 내 채널 안의 패턴에서 찾습니다.  
**간결한 요약** — 복잡한 수치 대신, 핵심만 골라 이해하기 쉽게 전달합니다.
