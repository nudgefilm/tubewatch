# TubeWatch — First User Test Checklist

## 테스트 목적

외부 사용자가 TubeWatch를 처음 접했을 때 주요 흐름이 정상적으로 작동하는지 검증합니다.

---

## 1. 랜딩 페이지 명확성

- [ ] `/` 접속 시 랜딩 페이지가 정상 렌더링되는가
- [ ] 서비스가 무엇을 하는지 3초 내에 이해할 수 있는가
- [ ] "내 채널 분석하기" CTA 버튼이 `/channels`로 이동하는가
- [ ] 모바일에서 레이아웃이 깨지지 않는가

## 2. 채널 등록 흐름

- [ ] `/channels` 페이지에서 채널 등록 UI가 명확한가
- [ ] YouTube 채널 URL 입력 후 등록이 성공하는가
- [ ] 등록 후 채널 목록에 새 채널이 표시되는가
- [ ] 잘못된 URL 입력 시 에러 메시지가 명확한가

## 3. 분석 요청 성공

- [ ] `/analysis/[channelId]`에서 "분석 요청" 버튼이 정상 작동하는가
- [ ] 분석 요청 후 상태가 "분석 진행 중"으로 변경되는가
- [ ] 분석 진행 중 스켈레톤 로딩 UI가 표시되는가
- [ ] 분석 완료 후 새로고침 시 리포트가 정상 렌더링되는가
- [ ] 72시간 쿨다운이 정상 적용되는가

## 4. 리포트 가독성

- [ ] Channel Metrics 카드가 정상 표시되는가
- [ ] Channel Score 바 차트가 정상 렌더링되는가
- [ ] Benchmark Radar 차트가 정상 표시되는가
- [ ] AI Insights 각 섹션이 빈 상태 처리를 포함해 정상 렌더링되는가
- [ ] Next Trend 섹션이 정상 표시되는가
- [ ] 모바일에서 리포트 레이아웃이 읽기 쉬운가

## 5. 인사이트 유용성

- [ ] 채널 요약이 채널 특성을 반영하는가
- [ ] 강점/약점이 메트릭 수치를 인용하는가
- [ ] 성장 액션 플랜이 실행 가능한 내용인가
- [ ] 추천 콘텐츠 주제가 채널과 관련이 있는가
- [ ] 인사이트에 과도하게 추상적인 표현이 없는가

## 6. 신뢰도 지표 이해

- [ ] 신뢰도 뱃지가 헤더에 정상 표시되는가
- [ ] 신뢰도 수치(점수)가 뱃지에 표시되는가
- [ ] hover 시 신뢰도 근거 tooltip이 나타나는가
- [ ] "신뢰도 높음/보통/낮음" 라벨이 직관적인가

## 7. 첫 사용자 가이드

- [ ] 분석 결과가 없을 때 FirstAnalysisGuide가 표시되는가
- [ ] 가이드 내용이 리포트 구성을 이해하는 데 도움이 되는가
- [ ] 닫기 버튼이 정상 작동하는가
- [ ] 채널 없음 상태에서 온보딩 메시지가 명확한가
